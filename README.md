# Velocity X

Projeto desenvolvido para o Trabalho Prático 1 da disciplina de Desenvolvimento de Interfaces Web.

---

## Sobre o projeto

O Velocity X é um site futurista e responsivo sobre supercarros de luxo e alta performance.  
A aplicação apresenta informações dinâmicas utilizando JavaScript e uma estrutura JSON local, sem necessidade de backend.

O projeto possui:

- Página inicial com destaque de carros
- Carrossel dinâmico
- Grid responsivo de cards
- Página de detalhes
- Galeria de imagens
- Navegação dinâmica utilizando Query String
- Layout responsivo para desktop, tablet e smartphone
- Interface Dark Futuristic UI
